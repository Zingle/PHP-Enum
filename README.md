# PHP Enum

Provides a simple enum like interface for constants. 

## Install

Via composer

~~~ bash
$ composer require zingle-com/php-enum
~~~

## Requirements

- PHP 7.1

## Testing

~~~ bash
$ vendor/bin/phpunit
~~~
